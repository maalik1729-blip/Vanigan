# Changelog

All notable changes to this project will be documented in this file.

### Added
- Initial release of Vanigan WhatsApp Bot
- Multi-language support (English, Tamil, Hindi)
- Business directory with 9 categories
- Organizer network directory
- Member directory
- Business registration flow
- Photo upload support
- Location sharing
- Subscription plans (Monthly, Yearly, Lifetime)
- Local news by district/assembly
- State machine conversation flow
- Global commands (MENU, BACK)
- RESTful API endpoints
- MongoDB database integration
- WhatsApp Cloud API integration
- Rate limiting
- Security middleware (Helmet, CORS)
- Logging system
- Error handling
- Input validation
- Database seeding script
- Comprehensive documentation

- 9 business categories with subcategories
